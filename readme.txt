Tiny plugin which improves UI styles for Contact Form 7

== Description ==

Better UI for Contact Form 7 improves Contact Form 7 user interface - loaders, alerts and input labels. 
You can choose type, size and color for the loader, while other styles are replaced without ability to edit them. 
Alert styles are replaced with [Bootstrap 4 alerts](https://getbootstrap.com/docs/4.3/components/alerts/).
Loaders are originally from [loading.io](https://loading.io/css/).

== Installation ==

1. Upload the entire `cf7-better-ui` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'CF7 Better UI' menu in your WordPress admin panel.
